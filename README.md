### 🚀 Retos Algorítmicos y Estadísticos Superados

El mayor desafío de este periodo académico radicó en dar el salto cualitativo desde la **estadística descriptiva** hacia la **inferencia paramétrica**, apalancándome en abstracciones avanzadas de programación dentro de *Google Colab*. 

Al integrar librerías como `scipy.stats` y `statsmodels` en Python, asimilé un principio fundamental para un ingeniero: el software no es una "caja negra", sino un conjunto de estructuras lógicas gobernadas por un rigor matemático estricto.
